## Typing stubs for peewee

This is a [type stub package](https://typing.python.org/en/latest/tutorials/external_libraries.html)
for the [`peewee`](https://github.com/coleifer/peewee) package. It can be used by type checkers
to check code that uses `peewee`. This version of
`types-peewee` aims to provide accurate annotations for
`peewee==4.0.0`.

This stub package is marked as [partial](https://typing.python.org/en/latest/spec/distributing.html#partial-stub-packages).
If you find that annotations are missing, feel free to contribute and help complete them.


This package is part of the [typeshed project](https://github.com/python/typeshed).
All fixes for types and metadata should be contributed there.
See [the README](https://github.com/python/typeshed/blob/main/README.md)
for more details. The source for this package can be found in the
[`stubs/peewee`](https://github.com/python/typeshed/tree/main/stubs/peewee)
directory.

This package was tested with the following type checkers:
* [mypy](https://github.com/python/mypy/) 1.19.1
* [pyright](https://github.com/microsoft/pyright) 1.1.408

[`24827c4590595507c5542ecea5c0850b070df4fc`](https://github.com/python/typeshed/commit/24827c4590595507c5542ecea5c0850b070df4fc).
