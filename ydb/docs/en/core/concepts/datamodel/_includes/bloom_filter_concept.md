A [Bloom filter](https://en.wikipedia.org/wiki/Bloom_filter) is a probabilistic data structure: for a given value it can tell you that the value is definitely not in the set, or that it may be present ([false positives](https://en.wikipedia.org/wiki/Bloom_filter#Probability_of_false_positives) are possible). When reading a table, the filter lets the storage layer skip data fragments that cannot contain the requested value, reducing the amount of data actually read.
