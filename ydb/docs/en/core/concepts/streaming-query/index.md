# Streaming data processing

This section describes the key concepts of stream processing ( [stream processing](https://en.wikipedia.org/wiki/Stream_processing)) in {{ ydb-short-name }}:

- [Streaming queries](streaming-query.md) — a type of query for continuous processing of incoming events. Describes data sources and sinks, delivery guarantees, limitations, and query management.
- [Watermarks](watermarks.md) — a mechanism for tracking the progress of event processing over time.
