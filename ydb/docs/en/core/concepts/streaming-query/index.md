# Stream data processing

This section describes the key concepts of [stream processing](https://en.wikipedia.org/wiki/Stream_processing) in {{ ydb-short-name }}:

- [Streaming queries](streaming-query.md) — a query type for continuous processing of incoming events. Covers data sources and sinks, delivery guarantees, limitations, and query management.
- [Watermarks](watermarks.md) — a mechanism for tracking event-processing progress by time.
