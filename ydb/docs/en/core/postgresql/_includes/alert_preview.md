
{% note alert %}

PostgreSQL wire protocol (pgwire) and PostgreSQL SQL dialect support (including the `--!syntax_pg` marker, the `enable_pg_syntax` feature flag, and the `ydb tools pg-convert` command) were **removed** from {{ ydb-short-name }}.

The pages in this section are kept for reference. To migrate from PostgreSQL, use [YQL](../yql/reference/index.md), the [SQL dialect converter](../integrations/sql-dialect-converter.md), or [data migration tools](../integrations/data-migration/index.md).

{% endnote %}
