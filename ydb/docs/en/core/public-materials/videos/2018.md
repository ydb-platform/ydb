1. [{{ ydb-name }}](https://youtu.be/Kr6WIYPts8I?t=8558): a platform for distributed storage systems that are latency-critical. A story about {{ ydb-short-name }} as a platform on which you can build various data storage and processing systems. Built on the {{ ydb-short-name }} platform are:

* LogBroker (an analog of Apache Kafka).
* Real-time Map Reduce (data stream processing).
* Time series storage in a monitoring system.

1. [{{ ydb-name }}](https://youtu.be/Kr6WIYPts8I?t=10550): Distributed SQL database. A story about {{ ydb-short-name }} as a database. From the talk you will learn:

* What a "tablet" is and what properties it has.
* How distributed transactions work in {{ ydb-short-name }} and what deterministic transactions are.
* What YQL is and how it differs from SQL.

1. [{{ ydb-name }}](https://youtu.be/Kr6WIYPts8I?t=12861): a network block device. A story about Network Block Store — a virtual disk service that powers all virtual machines in Yandex Cloud. The talk covers in detail:

* File System structure (File/directory, Metadata operations, Block index).
* Block Device structure (Data block storage, interaction interface).
* Working with QEMU/KVM (Virtio queues, CPU virtualization, paravirtualization mode).
