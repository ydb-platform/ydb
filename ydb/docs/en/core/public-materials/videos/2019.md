## {{ ydb-name }} — how we ensure fault tolerance {#2019-conf-hl-faul-tol}

{% include notitle [database_internals_tag](../_includes/tags.md#database_internals) %}

{{ ydb-short-name }} is a horizontally scalable geo-distributed fault-tolerant DBMS that withstands failures of disks, servers, racks, and data centers without compromising consistency. ## {{ ydb-name }} — an effective alternative to traditional NoSQL solutions {#2019-conf-yascale-ydb-nosql-alt}
