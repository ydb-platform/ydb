## News {{ ydb-name }} — overview of new features {#2020-webinar-ydb-news}

{% include notitle [overview_tag](../_includes/tags.md#overview) %}

[ {{ team.kovalenko.name }} ]({{ team.kovalenko.profile }}) ( {{ team.kovalenko.position }} ) provided an overview of new database features: working with JSON, system views, and nodeJS SDK. Previously, Yandex Cloud users could only access Yandex Database from virtual machines on the platform. Now it's possible via the internet.

From the webinar you will learn about:

* Publishing the driver for the Jäger trace collection and analysis system in Open Source.
* Enhanced capabilities for working with JSON in YQL.
* System views for query performance diagnostics.
* Publishing the JavaScript SDK in Open Source.
