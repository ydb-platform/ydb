# DROP TOPIC

`DROP TOPIC` deletes the specified [topic](../../../concepts/topic.md).

## Syntax

```yql
DROP TOPIC <topic_path>;
```

## Examples

The following command will delete the topic named `my_topic`:

```yql
DROP TOPIC my_topic;
```
