[Фильтр Блума](https://ru.wikipedia.org/wiki/Фильтр_Блума) — вероятностная структура данных, которая позволяет проверять принадлежность элемента множеству. Возможны [ложноположительные срабатывания](https://ru.wikipedia.org/wiki/Фильтр_Блума#Вероятность_ложноположительного_срабатывания) (элемента в множестве нет, но фильтр сообщает, что он может быть), [ложноотрицательных](https://ru.wikipedia.org/wiki/Фильтр_Блума) нет: если хотя бы один из проверяемых битов равен нулю, элемент в множестве отсутствует.

При чтении таблицы фильтр позволяет пропускать фрагменты данных, в которых искомое значение точно не встречается, и сокращать объём фактически считываемых данных.
