[Фильтр Блума](https://ru.wikipedia.org/wiki/Фильтр_Блума) — вероятностная структура данных: для заданного значения она может однозначно сказать, что значение точно отсутствует в наборе, либо что оно может присутствовать (возможны [ложноположительные срабатывания](https://ru.wikipedia.org/wiki/Фильтр_Блума#Вероятность_ложных_срабатываний)). При чтении таблицы фильтр позволяет пропускать фрагменты данных, в которых искомое значение гарантированно не встречается, и тем самым сокращать объём фактически считываемых данных.
