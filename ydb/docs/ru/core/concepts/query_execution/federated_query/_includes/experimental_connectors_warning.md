Внешние коннекторы являются экспериментальной возможностью {{ ydb-full-name }}. Для работы с внешними СУБД через коннекторы необходимо развернуть [fq-connector-go](../../../../devops/deployment-options/manual/federated-queries/connector-deployment.md#fq-connector-go) и явно включить соответствующие источники в конфигурации кластера {{ ydb-short-name }}. Функциональность может измениться, поэтому использование в производственной среде не рекомендуется без тщательного тестирования.
