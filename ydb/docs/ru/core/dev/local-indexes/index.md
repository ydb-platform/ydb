# Локальные индексы

[Локальные индексы](../concepts/query_execution/local_indexes.md) — вспомогательные структуры, которые хранятся вместе с данными таблицы и применяются при чтении на стороне хранилища. В отличие от глобальных индексов, они не материализуют отдельную индексную таблицу и не подключаются в запросе через `VIEW`.

В этом разделе — практические материалы по видам локальных индексов:

* [{#T}](../bloom-skip-indexes.md) — Блум-индексы для ускорения селективных запросов за счёт пропуска фрагментов данных.
