# Обзор

При выполнении пользовательского запроса {{ ydb-short-name }} фиксирует статистику его исполнения, такую как время работы отдельных шагов, объём переданных данных, задержки в ожидании данных. В других частях документации вы можете встретить использование термина `статистика` в другом контексте, например, статистика, характеризующая источники данных (таблицы). Это тесно связанные понятия и для выбора лучшенго плана оптимизатор опирается на статистику источников данных. Но в данном разделе под статистикой будет пониматься именно статистика времени выполнения, которая соответствует фактически полученным результатам измерений для данного конкретного запроса.

Эти измерения помогают исследовать план выполнения запроса, находить и устранять узкие места. Например, выбирая более оптимальные решения (изменяя текст запроса). В результате можно существенно сократить общее время работы запроса. Особенно это проявляется в [аналитических (OLAP) запросах](../../concepts/datamodel/table.md).

Статистика времени выполнения доступна вместе с планом запроса в [формате JSON](https://ru.wikipedia.org/wiki/JSON). Большое количество метрик делает их анализ и понимание достаточно сложным. Поэтому была разработана визуализация в [графическом формате SVG](https://ru.wikipedia.org/wiki/SVG). В данном разделе содержится информация об этом визуальном формате.

В качестве примера в данном разделе используется набор данных от [промышленного бенчмарка TPC-H scale 100](../../reference/ydb-cli/workload-tpch.md). Описание структуры таблиц и связей между ними вы можете найти в спецификации данного стандарта. Для иллюстрации мы использовали некоторые адаптированные из бенчмарка запросы, а также оригинальные, текст которых был специально подготовлен для иллюстрации материала. Текст каждого запроса приведён рядом с его планом выполнения.

Формат графического представления спроектирован таким образом, чтобы акцентировать внимание на наиболее нужных и полезных элементах. При этом опытный разработчик сможет получить из него больше информации, обращая внимание на дополнительные элементы и их свойства.
