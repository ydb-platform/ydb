# Добавление нового диска в кластер {{ ydb-short-name }}

В данном руководстве описан процесс добавления нового диска в кластер {{ ydb-short-name }}, развернутый в соответствии с топологией `3-nodes-mirror-3-dc`, с помощью Ansible.

## Предварительные требования

Перед началом работы убедитесь, что выполнены условия:

- Кластер {{ ydb-short-name }} развернут по топологии `3-nodes-mirror-3-dc`.

- На каждом сервере кластера установлен новый жесткий диск `/dev/vde`.
