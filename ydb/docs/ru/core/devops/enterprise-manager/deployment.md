# Запуск развёртывания {#deployment}

После выполнения всех подготовительных шагов можно приступить к развёртыванию {{ ydb-short-name }} Enterprise Manager.

## Автоматическое развёртывание {#automatic}

Перейдите в директорию `examples` и запустите скрипт установки:

```bash
cd examples
./install.sh
```

Скрипт `install.sh` выполнит все необходимые шаги для развёртывания {{ ydb-short-name }} Enterprise Manager на указанных в инвентаре хостах.

## Ручное развёртывание {#manual}

Если вам требуется больше контроля над процессом установки, вы можете запустить Ansible-плейбук вручную:

```bash
ansible-playbook ydb_platform.ydb_em.initial_setup
```

При возникновении ошибок подключения проверьте:
* Правильность указанного endpoint базы данных {{ ydb-short-name }}.
* Наличие сетевой связности между управляющей машиной и хостами кластера.
* Корректность TLS-сертификатов.
* SSH-доступ ко всем хостам из инвентаря.
