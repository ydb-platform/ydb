# Начало работы {#usage}

После успешного развёртывания {{ ydb-short-name }} Enterprise Manager вы можете начать работу с системой через веб-интерфейс.

## Доступ к веб-интерфейсу {#web-ui}

Откройте в браузере следующий адрес:

```
https://<FQDN_хоста_из_группы_ydb_em>:8789/ui/clusters
```

Где `<FQDN_хоста_из_группы_ydb_em>` — полное доменное имя любого хоста из группы `ydb_em` в Ansible-инвентаре.

{% note tip %}

Например, если в группе `ydb_em` указан хост `ydb-node01.ru-central1.internal`, адрес веб-интерфейса будет:

```
https://ydb-node01.ru-central1.internal:8789/ui/clusters
```

{% endnote %}
