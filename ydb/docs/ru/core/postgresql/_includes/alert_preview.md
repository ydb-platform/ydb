{% note warning %}

Поддержка синтаксиса PostgreSQL в {{ ydb-name }} **находится в разработке**. Использовать её в production окружениях **не рекомендуется**. Слушатель сетевого протокола PostgreSQL (pgwire) по умолчанию отключён.

Запросы в PostgreSQL могут исполняться до нескольких раз медленнее по сравнению с аналогичными запросами на YQL.

Основной сценарий, который можно тестировать — выполнение аналитических запросов к хранимым в {{ ydb-name }} данным.
