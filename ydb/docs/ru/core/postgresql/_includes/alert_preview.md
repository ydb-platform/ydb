{% note warning %}

Поддержка синтаксиса PostgreSQL в {{ ydb-name }} **находится в разработке**. Использовать её в production окружениях **не рекомендуется**. Слушатель сетевого протокола PostgreSQL (pgwire) на узлах `ydbd` **по умолчанию отключён** и не должен включаться на production-кластерах. Для локального тестирования используйте [локальный Docker-образ](../reference/docker/start.md), в котором pgwire включён по умолчанию.

Запросы в PostgreSQL могут исполняться до нескольких раз медленнее по сравнению с аналогичными запросами на YQL.

Основной сценарий, который можно тестировать — выполнение аналитических запросов к хранимым в {{ ydb-name }} данным.

{% endnote %}
