{% note warning %}

Поддержка синтаксиса PostgreSQL в {{ ydb-name }} **находится в разработке**. Использовать её в production окружениях **не рекомендуется**. Слушатель сетевого протокола PostgreSQL (pgwire) на узлах `ydbd` и в локальном Docker-образе **по умолчанию отключён** и не должен включаться на production-кластерах. Для локального тестирования задайте `YDB_EXPERIMENTAL_PG=1` при запуске контейнера.

Запросы в PostgreSQL могут исполняться до нескольких раз медленнее по сравнению с аналогичными запросами на YQL.

Основной сценарий, который можно тестировать — выполнение аналитических запросов к хранимым в {{ ydb-name }} данным.

{% endnote %}
