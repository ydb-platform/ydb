Команда выгрузки на файловую систему требует указания пути, доступного со стороны серверных хостов {{ ydb-short-name }}. Так как выгрузка производится в асинхронном режиме сервером {{ ydb-short-name }}, указанный путь должен быть доступен для записи со стороны всех узлов кластера. В случае сетевой файловой системы (NFS) это означает, что файловая система должна быть смонтирована на всех узлах.

`--fs-path PATH`: Обязательный параметр. Абсолютный путь в файловой системе, куда будут сохранены файлы выгрузки. В случае NFS здесь указывается полный путь к монтированной директории.
