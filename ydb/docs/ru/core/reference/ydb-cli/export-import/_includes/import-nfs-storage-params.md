Команда загрузки из файловой системы требует указания пути, доступного со стороны серверных хостов {{ ydb-short-name }}. Так как загрузка производится в асинхронном режиме сервером {{ ydb-short-name }}, указанный путь должен быть доступен для чтения со стороны всех узлов кластера. В случае сетевой файловой системы (NFS) это означает, что файловая система должна быть смонтирована на всех узлах.

`--fs-path PATH`: Обязательный параметр. Абсолютный путь в файловой системе, где расположены файлы с данными объектов YDB. В случае NFS здесь указывается полный путь к монтированной директории.
