{% note info %}

Режим `import` позволяет максимально задействовать ресурсы и быстро восстановить индексы, либо наоборот — ограничить ресурсы и скопировать индексные таблицы с минимальным влиянием на пользовательскую нагрузку. Для ограничения ресурсов импорта используется [очередь `queue_restore`](../../../configuration/resource_broker_config.md) брокера ресурсов.

{% endnote %}
