# Совместимость и миграция

Версии формата (согласно RFC 0115)

- Схема и версионирование metadata.json и mapping.json.

Правила восстановления между версиями

- Новые версии восстановления должны принимать старые версии форматов, если это документировано.

Миграция с устаревших бэкапов

- Переход от tools dump/restore к коллекциям; проверяйте результаты и целостность.

Заметки

- Сохраняйте действительность цепочек при применении политик хранения; не удаляйте полные бэкапы с зависимыми инкрементальными.
