# Современная команда ydb sql вместо устаревшей ydb yql

## Проблема

Использование современной команды `ydb sql` вместо устаревшей `ydb yql` обеспечивает работу через современный и развиваемый gRPC-сервис Query 

## Решение

Используйте команду `ydb sql`, которая работает через современный и развиваемый gRPC-сервис Query
