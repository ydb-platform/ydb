# Оптимизация кэша запросов под требования приложения

## Проблема

Дефолтные серверные настройки кэша запросов в {{ ydb-short-name }} могут быть неоптимальными для конкретного использования приложения. Это приводит к быстрому вымыванию кэша запросов и увеличению нагрузки на CPU из-за повторных компиляций одинаковых запросов.

## Решение

Перенастроить размер кэша запросов в настройках узла {{ ydb-short-name }}.
