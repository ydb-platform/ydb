# Профилактика возможных проблем

В этом разделе собраны практические заметки по работе с {{ ydb-short-name }}: типичные ошибки, ограничения и рекомендуемые подходы. Материалы сгруппированы по темам; там, где это уместно, внутри статей приведены «плохие» и «хорошие» примеры кода.

## Разделы

* [{#T}](balancer/index.md) — балансировка и маршрутизация запросов
* [{#T}](retry/index.md) — повторные попытки и обработка ошибок
* [{#T}](tables/index.md) — схема таблиц, индексы, партиционирование
* [{#T}](query/index.md) — YQL-запросы и производительность
* [{#T}](tx/index.md) — транзакции
* [{#T}](topics/index.md) — топики, продьюсеры, консьюмеры и интеграция через Kafka API
* [{#T}](common/index.md) — общие рекомендации (gRPC, тесты и т.д.)
* [{#T}](cli/index.md) — {{ ydb-short-name }} CLI
* [{#T}](configuration/index.md) — конфигурация кластера

## Соглашения

Примеры кода оформлены в виде раскрывающихся блоков «Плохой пример» и «Хороший пример». Если для одной темы приведено несколько языков, они объединены вкладками.

Не во всех статьях есть исполняемые примеры — отсутствие кода не означает, что тема неактуальна.
