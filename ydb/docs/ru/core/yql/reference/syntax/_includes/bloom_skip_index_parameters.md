Параметры Bloom skip индексов задаются в секции `WITH (...)`:

* `bloom_filter`
  * `false_positive_probability` — целевая вероятность ложноположительного срабатывания фильтра (диапазон `(0, 1)`). Если параметр не указан, используется значение по умолчанию: `0.1` для колоночных таблиц и `0.0001` для строковых.
* `bloom_ngram_filter` (для строковых колонок)
  * `ngram_size` — размер n-граммы, целое число от `3` до `8` (по умолчанию `3`).
  * `false_positive_probability` — целевая вероятность ложноположительного срабатывания (диапазон `(0, 1)`; по умолчанию `0.1`).
  * `case_sensitive` — учёт регистра при построении n-грамм: `true` или `false` (по умолчанию `true`).
