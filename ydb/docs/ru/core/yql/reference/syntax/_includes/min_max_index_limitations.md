* Поддерживаются только для [колоночных таблиц](../../../../concepts/glossary.md#column-oriented-table).
* В `ON (...)` должна быть указана ровно одна колонка.
* `COVER (...)` и дополнительные колонки данных не поддерживаются.
* Специфичные параметры `WITH (...)` не поддерживаются.
* `ALTER INDEX` для min_max-индекса не поддерживается.
* Колонки типов `Json` и `JsonDocument` не поддерживаются.
