# DROP GROUP

Команда `DROP GROUP` удаляет указанную [группу](../../../concepts/glossary.md#access-group). Для одного оператора вы можете указать несколько групп.

Синтаксис:

```yql
DROP GROUP [ IF EXISTS ] group_name [, ...]
```

* `IF EXISTS` — не выводить ошибку, если группа не существует.
* `group_name` — имя группы, которую нужно удалить. Может содержать только строчные буквы латинского алфавита (a–z), цифры (0–9). Имя не должно быть пустым.

## Примеры

### Успешно выполненные команды

Удаление группы:

```yql
DROP GROUP group1;
```

```yql
DROP GROUP IF EXISTS group1;
```

### Обработка ошибок

Корректно обработанные ошибки:

| Ситуация | Сообщение об ошибке |
|----------|---------------------|
| Удаление несуществующей группы | Group not found |
