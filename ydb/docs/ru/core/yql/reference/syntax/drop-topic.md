# DROP TOPIC

С помощью оператора `DROP TOPIC` можно удалить [топик](../../../concepts/datamodel/topic.md).

## Синтаксис

```yql
DROP TOPIC topic_path;
```

## Примеры

Следующая команда удалит топик с именем `my_topic`:

```yql
DROP TOPIC `my_topic`;
```
