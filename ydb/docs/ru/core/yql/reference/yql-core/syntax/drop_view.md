# DROP VIEW

DROP VIEW - удалить представление.

## Синтаксис

```sql
DROP VIEW имя
```

### Параметры

* `имя`

Имя представления, подлежащего удалению.

## Примеры

Следующая команда удалит представление со списком современных сериалов:

```sql
DROP VIEW recent_series;
```
