## Changes in built-in functions

Added the `LinearDestroy` function for consuming instances of linear types when their value is no longer needed.

## Changes in DateTime module {#datetime-module}

Introduced a new flag for `Datetime::Format` to write timezone offset with a colon.
Introduced a new specifier for both `Datetime::Parse` and `Datetime::Parse64` to read timezone offset.

## Changes in the Yson module {#yson-module}

Added the `Yson::Iterate` function for traversing a subtree of Yson nodes.
Added `Yson::As*` and `Yson::TryAs*` functions to cast a Yson node to a given type.
Added functions for performing [in-place modifications](../udf/list/yson.md#yson-modify) of Yson nodes.
