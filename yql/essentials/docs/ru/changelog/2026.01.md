## Изменения во встроенных функциях

Добавлена функция `AsOptionalType` для для идемпотентного оборачивания типа в Optional.
Добавлена функция `PositionOf` для получения позиции узлов.
Добавлена функция `WithIssue` для добавления в стек валидации пользовательского фрейма с диагностикой.

## Изменения в модуле Yson {#yson-module}

Добавлены функции `Yson::IsUtf8`, `Yson::AsUtf8` и `Yson::TryUtf8` для работы с узлами, которые гарантированно являются валидными Utf8 строками.
Также эти узлы используются для оптимизации работы функции `Yson::ConvertTo` если используется тип `Utf8`.
